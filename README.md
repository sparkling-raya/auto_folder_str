**USAGE**

Run this the first thing when you want to create a project folder

`bash create_structure.sh <your-project>`

Folder structure
```
/project_name/
  ├── data/
  │   ├── raw/
  │   └── processed/
  ├── src/
  │   ├── scripts/
  │   ├── functions/
  │   ├── notebooks/
  │   └── main.py
  ├── results/
  │   ├── figures/
  │   ├── reports/
  │   └── tables/
  ├── docs/
  │   └── index.md
  ├── env/
  ├── logs/
  └── tests/
  │   └── test_main.py
  ├── LICENSE
  └── README.md
```
